Attachment Drag and Drop

Load attachments directly via drag and drop into the form view. You can upload single files or whole folders via drag and drop. Uploaded files are automatically assigned to the current record.

Binary Widgets Drag and Drop

This module extends the existing binary widgets to add drag and drop capability. The function is only active during edit mode and only works for single files, with the exception of the multi attachment widget. With this widget you can add multiple files or whole folders via drag and drop.

Mail Drag and Drop

Load attachments directly via drag and drop into the mail chatter. You can upload single files or whole folders via drag and drop. Uploaded files are automatically assigned to the current message.
